# MSJC — Portfólio (React + Vite)

Aplicação de portfólio pessoal focada em apresentar perfil profissional, experiência e stack principal com uma UI responsiva e performática.

![Preview da aplicação](./src/assets/psmsjc-home.png)

## Demo

- Web: https://myspace-julio-chiuchi.vercel.app

## O que este projeto demonstra

- Componentização e UI responsiva com React + TailwindCSS
- Boas práticas com TypeScript em um setup moderno (Vite)
- Organização por componentes e páginas com foco em manutenção

## Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- ESLint (Rocketseat config)

## Como rodar localmente

Requisitos:

- Node.js 22+

Instalação e execução:

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
